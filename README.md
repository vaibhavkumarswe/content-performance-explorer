# Content Performance Explorer

A React-based dashboard for exploring content performance metrics, built with TypeScript and Material-UI.

## Tech Stack

- **Frontend**: React 19, TypeScript
- **UI Library**: Material-UI (MUI) with Emotion
- **Charts**: Recharts
- **Routing**: React Router
- **Build Tool**: Vite
- **Linting**: ESLint

## Prerequisites

- Node.js (version 18 or higher)
- npm (comes with Node.js)

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/vaibhavkumarswe/content-performance-explorer
   cd content-performance-explorer
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

## Running the Application

Start the development server:

```bash
npm run dev
```

The app will be available at `http://localhost:5173` (default Vite port).

## Building for Production

Build the application:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Linting

Run the linter:

```bash
npm run lint
```
